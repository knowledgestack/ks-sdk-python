# NetworkClass

Egress reachability class for an API connection.  PUBLIC is SSRF-guarded (private/internal ranges blocked); INTERNAL allows private/cluster targets and bypasses the range check.

## Enum

* `PUBLIC` (value: `'PUBLIC'`)

* `INTERNAL` (value: `'INTERNAL'`)
